This mod makes it so that you can teleport to a pylon without needing to be near a pylon. It does not make it ignore other conditions like npc count yet.

You can see this mod's source code here: https://github.com/Chik3r/TerrariaNoRangePylon

v0.2: Now able to change the amount of NPCs needed to be able to teleport to the pylon
v0.1: Initial release
